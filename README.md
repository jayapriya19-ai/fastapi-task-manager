# FastAPI Task Manager

A simple full-stack task manager application built using FastAPI.

## Features

- User Registration
- User Login
- JWT Authentication
- Create Tasks
- View Tasks
- Complete Tasks
- Delete Tasks

## Tech Stack

- FastAPI
- SQLite
- SQLAlchemy
- HTML/CSS/JavaScript

## Run Locally

```bash
pip install -r requirements.txt
uvicorn app.main:app --reload
